Solar System: 3D Scene with WebGL in p5.js

Goal: Create an interactive 3D scene using WebGL in p5.js


This project is a 3D model of a made up solar system. This is achieved, using all p5.js 3D WebGL primatives, p5.js' beginShape() function, simple lighting and camera functions, and an image for one of the planet's texture. Move the camera around by using right click, left click, and scrolling, and use the right and left arrow keys to jump between close up angles of all planets and star.
