Ponyta: 3D Wireframe Project

Goal: Using p5.js, build a complex model using basic three-dimensional parts


This project is a voxel model of the Pokemon, Ponyta. This is achieved, using 3 D cube primatives in conjunction with simple sorting algoriths for z axis-ordering. Use the v (voxel) and w (wireframe) keys to toggle display mode. Use arrow keys to rotate. Roatation achieved using 3D matrix rotations.
